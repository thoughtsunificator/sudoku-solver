# sudoku-solver

A grid based approach to solve sudokus

## Getting Started

### Usage

```javascript
import { Sudoku } from "@thoughtsunificator/sudoku-solver"
const puzzle = [
  [5,3,0,0,7,0,0,0,0],
  [6,0,0,1,9,5,0,0,0],
  [0,9,8,0,0,0,0,6,0],
  [8,0,0,0,6,0,0,0,3],
  [4,0,0,8,0,3,0,0,1],
  [7,0,0,0,2,0,0,0,6],
  [0,6,0,0,0,0,2,8,0],
  [0,0,0,4,1,9,0,0,5],
  [0,0,0,0,8,0,0,7,9]
]
const sudoku = new Sudoku(puzzle)
const solution = sudoku.solve()
```
